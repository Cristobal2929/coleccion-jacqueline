# Colección Jacqueline – Boutique de eventos

Esta es una web estática de una sola página que muestra el catálogo y los servicios de la boutique de eventos **Colección Jacqueline**.  

- **Diseño responsive**: se adapta a móviles, tablets y escritorio.  
- **Tipografía**: *Playfair Display* para los encabezados y *Lato* para el cuerpo del texto.  
- **Accesibilidad**: foco visible, respeta `prefers-reduced-motion`.  
- **Imágenes**: todas provienen de la carpeta `static/img/` (10 imágenes reales).  

## Estructura de archivos

- `index.html` – página única con CSS y JavaScript embebidos.  
- `README.md` – este documento con metadatos del proyecto.  

## Cómo usar

1. Copia la carpeta completa al servidor estático que prefieras.  
2. Asegúrate de que la ruta `static/img/` contenga los diez archivos de imagen indicados.  
3. Abre `index.html` en cualquier navegador.  

¡Listo! La web está preparada para presentar la boutique de forma elegante y moderna.
