# Colección Jacqueline

Sitio web estático de una boutique especializada en vestidos de gala, tocados personalizados y complementos para eventos especiales.

## Estructura de archivos

- **index.html** – Página única, autocontenida con CSS y JavaScript embebidos.
- **static/img/** – Carpeta con las 9 imágenes proporcionadas (`imagen_1.jpg` … `imagen_9.jpg`).

## Cómo usar

1. Coloca las imágenes en la ruta `static/img/` tal y como aparecen en el proyecto.
2. Si dispones de un número de WhatsApp, sustituye el marcador `[TU WHATSAPP]` en el archivo `index.html` por el enlace completo, por ejemplo: `https://wa.me/34XXXXXXXXX`.
3. Abre `index.html` en cualquier navegador. El sitio es totalmente responsive y respeta la preferencia de reducción de movimiento.

## Tecnologías

- HTML5
- CSS3 (Flex y Grid)
- JavaScript puro
- Google Fonts (Playfair Display & Montserrat)

## Accesibilidad

- Enfoque visible para todos los elementos interactivos.
- Compatibilidad con `prefers-reduced-motion`.
- Texto en español de España y precios en euros (no aplicable en este proyecto).
