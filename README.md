# Colección Jacqueline – Boutique de eventos

Esta es una página estática de una sola página (single‑page) que muestra el catálogo y los servicios de la boutique de eventos **Colección Jacqueline**.  

- **Sin frameworks ni build tools** – todo el CSS y JavaScript están incluidos en `index.html`.  
- **Responsive** y accesible: foco visible, `prefers-reduced-motion`, y navegación por teclado.  
- **Tipografía**: *Playfair Display* (títulos) y *Montserrat* (texto).  
- **Imágenes**: se usan las siete imágenes provistas en `static/img/`.  

Para publicar, simplemente sirve la carpeta que contiene `index.html` y `README.md` con cualquier servidor estático (GitHub Pages, Vercel, Netlify, etc.).
