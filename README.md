# Colección Jacqueline – Sitio estático

Este repositorio contiene la versión estática y autocontenida de la web de **Colección Jacqueline**, boutique de ropa y complementos para eventos especiales.

## Estructura de archivos

- `index.html` – Página única con todo el contenido, estilos y scripts necesarios.
- `static/img/` – Carpeta con las 9 imágenes proporcionadas (`imagen_1.jpg` … `imagen_9.jpg`).

## Cómo usar

1. Clona o descarga el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
3. La página es totalmente responsiva y funciona sin dependencias externas (excepto Google Fonts).

## Notas de diseño

- Tipografía: **Playfair Display** (títulos) + **Montserrat** (texto).
- Paleta de color principal: indigo‑purple (`#5b2c6f`).
- Enfoque en la elegancia y una única apuesta visual: la foto hero del vestido largo.
- Cumple con accesibilidad: foco visible, `prefers-reduced-motion`, y navegación por teclado.
