# Colección Jacqueline – Boutique de eventos

Esta es una página web estática de una sola página para **Colección Jacqueline**, boutique femenina especializada en ropa y complementos para eventos (vestidos largos, boda, chaquetones, tocados, estolas, bolsos y servicio de arreglos).

## Estructura de archivos

- `index.html` – Página única, autocontenida con CSS y JavaScript (sin frameworks).
- `static/img/` – Carpeta con las siete imágenes proporcionadas (`imagen_1.jpg` … `imagen_7.jpg`).

## Tecnologías usadas

- HTML5
- CSS3 (variables, grid, media queries)
- Google Fonts (Playfair Display y Montserrat)

## Cómo probar

1. Clona o descarga el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
3. La página es totalmente responsive y respeta la preferencia `prefers-reduced-motion`.

## Personalización

- Cambia los colores modificando las variables `--color-primary`, `--color-bg`, etc. en la sección `:root` del CSS.
- Sustituye el marcador `[TU WHATSAPP]` por el número de WhatsApp real cuando lo tengas disponible.
- Añade o elimina imágenes en la carpeta `static/img/` y actualiza los `<img>` correspondientes en `index.html`.
